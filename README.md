# Python Flask Accounting App
An Accounting Web App built by Python Flask

## Extensions
1. Flask-Script
2. Flask-Migrate
3. Flask-Mongoengine
4. Flask-Login
5. Flask-OAuthlib
6. Flask-Principal 
7. Flask-SQLAlchemy
8. Flask-WTF
9. Passlib
